# limpeza-posformatnodes
Aplicação em .batch para Windows, para limpeza, manutenção e otimização do sistema.

Observação muito Importante:
Esta ferramenta está sendo desenvolvido por um único desenvolvedor, e não é experiente.
As linhas de comando contidos no código, os quais você pode olhar livremente, são baseados scripts de outros desenvolvedores. Isso você pode ver no próprio código, pois mantivemos parte do nome dos criadores, até para os homenagear. Outros não foram citados, pois nem humanos são, são IAs que auxiliam na hora de corrigir e melhor este scritp, mas obrigado mesmo assim.
